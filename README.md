# Overview

The goal of this project is to create a comprehensive configuration guide and explain the underlying technologies in a simple, accessible way for all topics covered in the CCNA 200-301 v1.1 and CCNP ENCOR 350-401 v2 exams.

>[!NOTE]
>The topics are ordered progressively, starting with the simplest configurations and advancing to the most complex technologies, rather than following the official topic sequence of the CCNA and CCNP blueprints.
