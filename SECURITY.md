# Security Policy

## Supported Versions
Currently, the Cisco IOS Comprehensive Guide is in active development, and official versions have not yet been released. All security and accuracy updates are applied directly to the main branch.

## Reporting a Vulnerability
If you discover a security vulnerability or a critical flaw within this project, please do not open a public issue. Instead, report it privately by sending an email to:
**coieclub@outlook.com**

We take all reports seriously. You can expect an initial response and acknowledgment of your report within a maximum of **3 days**.
